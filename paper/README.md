# Paper — Faithful-by-Concept (MICAD 2026)

- **`main.tex`** — submission manuscript, **Springer Proceedings class (`svproc`)** — the class MICAD requires.
- **`DRAFT.md`** — the full-prose draft (easier to read/edit); `main.tex` is the LaTeX of the same content.

## Compile on Overleaf (recommended)
1. On the MICAD site, click **Download the Template@Overleaf** → **Open as Template** (creates an
   editable copy in your Overleaf account, with `svproc.cls` already present).
2. Replace the sample `author.tex` content with our **`main.tex`** (copy-paste the whole file),
   or upload `main.tex` and set it as the main document.
3. Upload the figure **`qualitative_melanoma.png`** (from `artifacts/figures/`, generated by
   `scripts/make_report.py`) to the project root — `main.tex` includes it as `qualitative_melanoma.png`.
4. Compile (pdfLaTeX).

## Compile locally (alternative)
Put `main.tex`, `qualitative_melanoma.png`, and the template's `styles/svproc.cls`
(+ `aliascnt.sty`, `remreset.sty`) in one folder, then `pdflatex main` twice.
